# Contributing to acy-1.0

## Proposing New Skills
1. Document technique in wiki/technique/{name}.md with YAML frontmatter
2. Run "evolve skills" to trigger gap analysis
3. Review proposed SKILL-*.md draft
4. Approve or provide feedback for modification

## Wiki Standards
- YAML frontmatter required on every page
- Use [[wiki/page-name]] for bi-directional links
- Include confidence score (1-5)
- Cite evidence files and proxy history

## Tool Integration
- Update SKILL-TOOLS.md with new tools
- Include installation command
- Add MCP invocation pattern if applicable
- Log tool failures in wiki

## Chain Recipes
- Document successful chains in SKILL-CHAIN.md
- Include step-by-step execution
- Specify CIA impact ratings
